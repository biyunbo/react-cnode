# react-cnode
基于webpack2 + react + react-router4 + redux + less + ES6 的React版cnode社区
------------------------------------------------------------------
<h3>下载</h3>
<pre><code>
git clone https://github.com/biyunbo/react-cnode.git
cd react-cnode
npm install yarn -g(如果没有yarn)
yarn(安装依赖)
</code></pre>
<h3>启动</h3>
<pre><code>
yarn start(开发环境访问：http://localhost:8888/)
</code></pre>
<h3>模块</h3>
<pre><code>
1.登录，退出
2.个人中心
3.列表，列表详情
4.发布主题
5.消息
</code></pre>
